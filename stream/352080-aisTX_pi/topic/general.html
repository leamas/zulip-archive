Fabian Olsthoorn (Nov 03 2022 at 11:12):

The payload(So the text in the message) can change, but the package around the payload cannot change as easly. So if we have !xxBBM,1,1,0,1,8,<Rh76039hmkqo08F2BE7P?@5>,0*4D

!xxBBM: There might be some other prefix that works, but this is the only one I found so far
1,1,0: Depends on if the full message needs mutiple messages to fit
1: Is the radio channel A or B. 1 for A, 2 for B
8: Is the message type. So can be 6 too for example, but then we'd have to add a destination MMSI to the payload.
Then for the payload I use https://www.e-navigation.nl/content/isrs-text-message for this. And then only the binary data part is the payload.
And lastly the fillbit and checksum which always change.
