Rick Gleason (Jun 15 2024 at 22:19):

Note to all Editors

Every page in the manual is intended to have an indexmenu header like this

{{indexmenu_n>6}}

This header determines the order of the namespace in the navigation menu on the right.

Someone has been removing these headers.

Please DO NOT DO THAT!!!
Thanks, Rick

PS: It may be your text editor that does this. I have noticed some weird changes that I think is due to the text editor. Please take care to notice these changes, and perhaps use a different editor.
