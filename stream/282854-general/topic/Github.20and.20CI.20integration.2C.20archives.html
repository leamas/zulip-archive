Alec Leamas (Dec 20 2022 at 08:47):

I have given Zulip some love trying to make it as usable as similar tools I have seen in other projects.

  • New stream opencpn-commits. Here, every new pushed commit to main opencpn shows up. Useful for everyone besides Dave ;)
  • The CI stream now contains reports from all CI builds. Useful but somewhat noisy.
  • I have started a job which archives our community at https://leamas.github.io/zulip-archive/. The primary purpose of this is to let the web crawlers index discussions so they can be found using google and friends. Besides this, the html interface is IMHO not that useful.
  • The resurrected stream comms is removed for the second time. Lately, it contained the commits now in opencpn-commits.

Alec Leamas (Jan 14 2023 at 13:43):

More love given to zulip. It now generates links in the same way as github. For example #2927, a link to an issue, PR or Discussion. Or 62b4c9932, a link to a commit. At least 8 hex digits is required for the commit.
